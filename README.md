# MSPM0G3507_core_board
 
>_注意：这是KiCAD工程！_

## 功能

* 板载135*240LCD屏幕
* DC电源接口
* 总电源拨动开关
* 12V、5V、3.3V电源排针接口
* TYPE-C供电接口
* 提供CAN-FD接口（未验证）
* 提供三个用户按键
* 提供两个板载LED
* 提供重启键、BSL键
* 引出串口（GH1.25 4P）、SWD接口（MX1.25 4P）

## 特点

* 支持3S锂电池供电
* 12V、5V电源具备TVS保护
* 板载外设可通过电阻使能
* 一键进入BSL
* CAN接口TVS保护
* TYPE-C接口500mA保险丝保护
* 芯片排针接口间距为100mil整倍数，方便洞洞板插接
